Objective of G-Tech program is to make the student competent in different areas of IT which includes Basic Computer Operations, Computerised Accounting, Desk Top Publishing, Web Site Designing. Computer Hardware & Networking, Digital Marketing. Student can choose optional module in semester-V with Web Development, System Programming with C, C++, Relational Database Management System (MySQL SQL Server), .Net Technology, Android App Development and specialisation in Animation. On completion of G-Tech program the student will be able to work as a Software Professional with an IT Company.
